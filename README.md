# Technology Stack

This project uses a diverse stack of technologies to achieve its goals. Below is a comprehensive list of the technologies included across all repositories:

- **Jupyter Notebook**: For interactive computing and data visualization.
- **C++**: Utilized for performance-critical applications.
- **C**: Foundational programming language used for system programming.
- **CMake**: A tool designed to build and manage the build process for software projects.
- **Makefile**: Used for defining build processes using the `make` build automation tool.
- **HTML**: The standard markup language for creating web pages.
- **CSS**: For styling HTML elements.
- **FastAPI**: A modern web framework for building APIs with Python, focused on speed and efficiency.
- **MongoDB**: A NoSQL database used for scalable data storage.
- **Redis**: An in-memory data structure store, used for caching and quick storage.

Additional technologies may be incorporated in the future as the project evolves.
